# 기능 목록

## 1. 게임 시작 및 초기 상태

- [x] 게임 시작 시 초기 스탯을 설정한다.
	- 체력: 100
	- 식량: 3
	- 감염도: 10
	- Day: 1
	- 치료 선택 횟수: 0
	- 구조 포인트: 0
- [x] 게임 시작 시 덱을 셔플하고 남은 카드 수를 20으로 표시한다.
- [ ] 초기 화면에서 카드 영역, 로딩 영역, 결과 화면을 숨긴다.
- [ ] 카드 뽑기 버튼과 포기 버튼을 활성화 상태로 표시한다.

## 2. 카드 덱 구성 및 관리

- [x] 카드 20장을 아래 비율로 구성한다.
	- 생존자 시체: 4장
	- 부상당한 군인: 4장
	- 임시 수술: 3장
	- 군용 차량 행렬: 3장
	- 오염된 웅덩이: 3장
	- 구조 트럭: 3장
- [x] 카드 뽑기 시 남은 카드 수를 1 감소시킨다.
- [x] 덱이 소진되면 자동으로 다시 셔플해 사용한다.

## 3. 카드 뽑기 및 선택 흐름

- [x] 카드 뽑기 버튼 클릭 시 카드 영역을 표시한다.
- [x] 카드 이름/설명/선택지 A/B를 화면에 출력한다.
- [ ] 카드가 열린 동안 카드 뽑기 버튼을 숨긴다.
- [ ] 선택지 클릭 시 2초 동안 로딩을 표시한다.
- [ ] 선택 처리 중 선택지 버튼을 비활성화한다.
- [ ] 선택 처리 완료 후 로그에 결과를 출력한다.
- [ ] 선택 처리 완료 후 Day를 1 증가시킨다.
- [ ] 선택 처리 완료 후 카드 영역을 숨기고 카드 뽑기 버튼을 다시 표시한다.

## 4. 카드 효과 적용

- [x] 생존자 시체
	- A: 식량 +3, 감염도 +8
	- B: 식량 +1
- [x] 부상당한 군인
	- A: 식량 -1, 감염도 -20, 치료 선택 횟수 +1
	- B: 체력 -10, 식량 +2
- [x] 임시 수술
	- A: 체력 -25, 감염도 -25, 치료 선택 횟수 +1
	- B: 체력 -5, 감염도 +10
- [x] 군용 차량 행렬
	- A: 구조 포인트 +1, 감염도 +8
	- B: 체력 +5
- [x] 오염된 웅덩이
	- A: 체력 +5, 감염도 +15
	- B: 체력 -10
- [x] 구조 트럭
	- A: 체력 -20, 구조 포인트 +1
	- B: 체력 +10

## 5. 턴 종료 후 기본 소모 규칙

- [ ] 카드 효과 적용 후 식량을 1 소비한다.
- [ ] 카드 효과 적용 후 감염도를 3 증가시킨다.
- [ ] 식량이 0인 상태에서 선택을 완료하면 체력을 10 추가 감소시킨다.
- [ ] 기아 피해 발생 시 로그에 안내 메시지를 출력한다.

## 6. 수치 경계값 처리

- [ ] 체력은 0 미만으로 내려가지 않도록 보정한다.
- [ ] 식량은 0 미만으로 내려가지 않도록 보정한다.
- [ ] 감염도는 0 미만으로 내려가지 않도록 보정한다.

## 7. 엔딩 판정

- [ ] 실패 엔딩을 성공 엔딩보다 우선 판정한다.
- [ ] 실패 엔딩
	- 체력 <= 0: 사망
	- 감염도 >= 100: 좀비화
- [ ] 성공 엔딩
	- 치료 선택 5회 이상: 치료 성공
	- 구조 포인트 3 이상 + Day > 10: 구조 성공
	- Day > 15: 생존 성공
- [ ] 플레이어가 포기 버튼을 누르면 즉시 포기 엔딩으로 종료한다.

## 8. 결과 화면 출력

- [ ] 게임 종료 시 결과 화면을 표시한다.
- [ ] 결과 화면에 아래 항목을 출력한다.
	- 생존 일수
	- 최종 체력
	- 최종 식량
	- 최종 감염도
	- 엔딩 종류

## 9. 재시작

- [ ] 게임 종료 후 다시 시작 버튼을 표시한다.
- [ ] 다시 시작 버튼 클릭 시 게임 화면으로 복귀한다.
- [ ] 스탯, Day, 치료 선택 횟수, 구조 포인트를 초기값으로 복원한다.
- [ ] 덱을 20장 상태로 초기화한다.
- [ ] 로그를 초기화한다.
- [ ] 재시작 후 선택지 버튼이 정상 동작하도록 상태를 초기화한다.
