# 기능 목록

## 게임 상태 관리

- [ ] 초기 상태 생성 (체력 100, 식량 3, 감염도 10, Day 1)
- [ ] 치료 횟수(healAttempts), 구조 포인트(rescuePoints) 추적
- [ ] 카드 덱 생성 및 셔플

## 카드 덱 시스템

- [ ] 20장 카드 덱 생성 (6종류, 분배표에 따라)
- [ ] Fisher-Yates 셔플 알고리즘
- [ ] 카드 뽑기 (덱에서 1장 pop)
- [ ] 덱 소진 시 자동 리셔플

## 카드 종류 및 선택지

- [ ] 좀비 무리 (4장): 맞서 싸운다 / 숨을 죽인다
- [ ] 약국 잔해 (4장): 안으로 들어간다 / 밖의 식량만 줍는다
- [ ] 빈 아파트 (3장): 문을 잠그고 쉰다 / 방마다 뒤진다
- [ ] 무장한 생존자 (3장): 식량을 건넨다 / 치료를 부탁한다
- [ ] 감염 발작 (3장): 이를 악물고 견딘다 / 감염 부위를 도려낸다
- [ ] 무전 신호 (3장): 위치를 송신한다 / 무시한다

## 선택지 처리

- [ ] 선택 후 2초(2000ms) 대기
- [ ] 선택에 따른 스탯 변화 적용
- [ ] 치료 선택 시 healAttempts 증가
- [ ] 구조 응답 시 rescuePoints 증가

## 일일 자동 효과

- [ ] 식량 1 소비 (최솟값 0)
- [ ] 감염도 3 증가
- [ ] 식량 0이면 체력 10 추가 감소

## 엔딩 조건

- [ ] 사망: 체력 <= 0
- [ ] 좀비화: 감염도 >= 100
- [ ] 치료 성공: 치료 선택 5회 누적
- [ ] 구조 성공: 구조 포인트 3 이상 + Day > 10
- [ ] 생존 성공: Day > 15
- [ ] 포기

## 엔딩 결과 출력

- [ ] 생존 일수, 최종 체력, 최종 식량, 최종 감염도, 엔딩 표시

## UI

- [ ] 현재 Day 및 스탯 표시
- [ ] 남은 카드 수 표시
- [ ] 카드 뽑기 버튼
- [ ] 카드 이름/설명 표시
- [ ] 선택지 A/B 버튼
- [ ] 선택 후 2초 대기 (로딩 표시)
- [ ] 게임 로그 표시
- [ ] 게임 종료 화면
- [ ] 다시 시작 버튼
- [ ] 포기 버튼

## 입력 유효성 검사

- [ ] 선택 처리 중 버튼 비활성화
- [ ] 게임 종료 후 입력 차단
