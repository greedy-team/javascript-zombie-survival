#기능 구현 목록

1. 초기화 및 세팅
    - 게임 시작 시 초기 스탯(체력 100, 식량 3, 감염도 10, Day 1) 설정

    - 6종류의 총 20장의 카드 데이터 생성

    - 초기 상단 스탯 화면 UI 업데이트

2. 카드 덱 시스템
    - 20장의 카드 뭉치를 무작위로 섞는 셔플 기능

    - 덱에서 카드 1장을 뽑는 기능

    - 20장을 모두 사용했을 때 자동으로 리셔플하는 기능

3. 게임 플레이 로직
    - '카드 뽑기' 버튼 클릭 시 화면 전환 (뽑기 영역 숨김 -> 카드 영역 표시)

    - 뽑은 카드의 정보(이름, 설명, 선택지 텍스트)를 화면에 렌더링

    - 선택지(A/B) 클릭 시 처리:

        - 선택지 클릭 시 즉시 로딩 화면 표시

        - setTimeout을 사용하여 2초 대기 후 결과 반영

    - 스탯 계산 및 반영:

        - 선택한 옵션에 따른 카드 효과를 우선 적용

        - 매 선택 후 식량 1 감소, 감염도 3 증가

        - 식량이 0인 상태에서 행동 시 체력을 10 추가 감소

    - 변화된 스탯을 상단바에 즉시 업데이트하고, 하단 로그창에 결과 메시지 추가

4. 엔딩 및 게임 종료 시스템
    - 매 턴 종료 후 아래 우선순위에 따라 엔딩 여부 판별

        - 사망: 체력 0 이하

        - 좀비화: 감염도 100 이상

        - 치료 성공: 치료 관련 선택 5회 누적

        - 구조 성공: 구조 포인트 3점 이상 이면서 10일 초과 생존

        - 생존 성공: 15일 버티기 성공

    - 달성한 엔딩 종류와 최종 생존 일수, 최종 스탯 결과 페이지 표시

    - '포기하기' 버튼 클릭 시 즉시 게임 종료 후, 현재까지의 결과 표시

    - '다시 시작' 버튼 클릭 시 모든 스탯과 덱 초기화 후, 초기 상태로 리셋
