<h1 align="middle">좀비 아포칼립스: 감염자의 이지선다</h1>

## 🔍 진행방식

- 미션은 **기능 요구사항, 프로그래밍 요구사항, 과제 진행 요구사항** 세 가지로 구성되어 있다.
- 세 개의 요구사항을 만족하기 위해 노력한다. 특히 기능을 구현하기 전에 기능 목록을 만들고, 기능 단위로 커밋 하는 방식으로 진행한다.

---

## 🎯 기능 요구사항

원인 불명의 감염 사태가 발생했다. 도시는 봉쇄되었고, 당신은 **이미 좀비에게 물렸다.**
완전히 좀비로 변하기 전에 **치료제를 완성**하거나, **15일을 버텨 구조대를 기다려야** 한다.

이 게임은 **웹 브라우저에서 실행**된다. 플레이어는 버튼을 클릭해 카드를 뽑고 선택하며, 모든 결과는 화면에 표시된다.

- 게임 시작 시 플레이어의 초기 상태는 **체력 100, 식량 3, 감염도 10**이다.
- 매일(Day) **카드 1장을 뽑고**, 카드의 **선택지 2개 중 1개를 고른다.**
- 선택지를 고른 후 **2초(2000ms)** 가 지난 후 결과가 화면에 표시된다.
- 매일 선택지를 고르면 **카드 효과가 먼저 적용**된 후, **식량이 1 소비**되며 **감염도가 3 증가**한다.
- 식량이 0인 상태에서 선택 시 **체력이 10 추가 감소**한다.
- 치료 선택과 구조 응답 횟수가 **화면 상단 스탯에 실시간으로 표시**된다.
- 체력은 **0 미만으로 내려가지 않는다.** (최솟값 0)
- 식량은 **0 미만으로 내려가지 않는다.** (최솟값 0)
- 감염도는 **0 미만으로 내려가지 않는다.** (최솟값 0, 0은 일시적 억제 상태로 완치가 아님)
- 플레이어는 언제든 **게임을 포기**할 수 있다.

### 🃏 카드 덱 규칙

20장의 카드로 구성된 덱을 셔플하여 사용한다. 매일 **1장을 뽑고 선택지를 고른다.**
덱이 소진되면 자동으로 리셔플된다.

| 카드             | 장수 | 선택지 A                                              | 선택지 B                                                       |
| ---------------- | ---- | ----------------------------------------------------- | -------------------------------------------------------------- |
| 🧟 생존자 시체   | 4    | 배낭째로 가져온다 (식량 +3, 감염도 +8)                | 겉에 있는 것만 집는다 (식량 +1)                                |
| 💊 부상당한 군인 | 4    | 식량을 건네고 약을 받는다 (식량 -1, 감염도 -20)       | 몸싸움 끝에 식량만 챙기고 떠난다 (체력 -10, 식량 +2)          |
| 🔪 임시 수술     | 3    | 감염 부위를 직접 도려낸다 (체력 -25, 감염도 -25)      | 이를 악물고 참는다 (체력 -5, 감염도 +10)                       |
| 🚗 군용 차량 행렬 | 3   | 뛰어나가 신호를 보낸다 (구조 포인트 +1, 감염도 +8)    | 몸을 낮추고 방향만 확인한다 (체력 +5)                          |
| 💧 오염된 웅덩이 | 3    | 그냥 마신다. 탈수보다 낫다 (체력 +5, 감염도 +15)      | 참는다. 빗물을 기다린다 (체력 -10)                             |
| 🚛 구조 트럭     | 3    | 전력으로 달려간다 (체력 -20, 구조 포인트 +1)          | 체력을 아끼고 쉰다 (체력 +10)                                  |

### 🏁 엔딩 조건

#### 실패 엔딩

- **체력 ≤ 0** → 사망
- **감염도 ≥ 100** → 좀비화

#### 성공 엔딩

- **치료 선택 5회 누적** → 치료 성공
  - 치료 도움: 💊 부상당한 군인 - 식량을 건네고 약을 받는다
  - 자가치료: 🔪 임시 수술 - 감염 부위를 직접 도려낸다
- **구조 포인트 3 이상 + Day > 10** → 구조 성공
- **Day > 15** → 생존 성공 (구조대 도착)

---

#### ⚠️ 엔딩 우선순위

여러 엔딩 조건이 동시에 충족될 경우, 실패 엔딩(사망 / 좀비화)을 성공 엔딩보다 우선 적용한다.

## ✅ 프로그래밍 요구사항

- `var`를 사용하지 않는다. `const`, `let`만 사용한다.
- 전역 변수를 사용하지 않는다.
- 외부 라이브러리를 사용하지 않고, 순수 Vanilla JS로만 구현한다.
- **Airbnb 자바스크립트 코드 컨벤션**을 지키면서 프로그래밍 한다.
- indent(인덴트, 들여쓰기) depth를 2가 넘지 않도록 구현한다.
  - 예를 들어 while문 안에 if문이 있으면 들여쓰기는 2이다.
  - 힌트: indent depth를 줄이는 좋은 방법은 함수(또는 메소드)를 분리하는 것이다.
- 함수(또는 메소드)가 한 가지 일만 하도록 최대한 작게 만들어라.
- `import` 문을 이용해 스크립트를 모듈화하고 불러올 수 있게 만든다.
- 함수(또는 메소드)의 길이가 15라인을 넘어가지 않도록 구현한다.

## ✔️ 테스트 실행 가이드

- 테스트 실행에 필요한 패키지 설치를 위해 `Node.js` 버전 `20` 이상이 필요하다.
- 다음 명령어를 입력해 패키지를 설치한다.

```
npm install
```

- 설치가 완료되었다면, VSCode의 **Go Live** 버튼을 클릭해 로컬 서버(`http://localhost:5500`)를 먼저 실행한다.

- 서버가 실행된 상태에서 다음 명령어를 입력해 테스트를 실행한다.

```
npm run test
```

- 모든 테스트가 pass한다면 성공!
